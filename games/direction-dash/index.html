HOW TO PLAY

  1. 01
    Grid reveals before GO!
    All arrows appear 1 second before the timer starts — use it to prepare.
  2. 02
    Match each highlighted cell
    Input the direction of the glowing cell — top-left to bottom-right.
  3. 03
    Wrong input = −1 second
    Every mistake drains the clock. Stay sharp.
  4. 04
    Two input modes
    Desktop: WASD keys or mouse swipe. Mobile: D-pad buttons or swipe screen.
